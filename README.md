# API Atlas

## Project Overview

API Atlas is a web application that allows users to explore and discover public APIs from various categories.
The application uses a local `data.json` file to display APIs in a clean and responsive interface.

Users can search, filter, and sort APIs easily.

---

## Features

* Fetch data from local JSON file
* Display APIs in card layout
* Search APIs by name or description
* Filter APIs by category
* Sort APIs (A to Z, Z to A)
* Fully responsive design (mobile, tablet, desktop)

---

## Technologies Used

* HTML
* CSS (Responsive Grid)
* JavaScript
* Fetch API
* Array Methods:

  * map()
  * filter()
  * sort()

## How to Run the Project

1. Clone the repository

   ```
   git clone <your-repo-link>
   ```

2. Open the project folder

3. Run using Live Server (recommended)

   * Right-click index.html
   * Click "Open with Live Server"

Note: Do not open directly using file://
