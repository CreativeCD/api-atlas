# API Atlas 🌐

## Project Overview

API Atlas is a web application that allows users to explore and discover public APIs from various categories. The application fetches real-time data from a public APIs directory and displays it in a clean and user-friendly interface. Users can search, filter, and sort APIs, mark favorites, and switch between dark and light mode.

This project demonstrates the use of JavaScript, Fetch API, Array Higher-Order Functions, responsive UI design, and local storage.

---

## Features

* Fetch data from a Public APIs API
* Display APIs in card layout
* Search APIs by name
* Filter APIs by category
* Sort APIs alphabetically
* Dark mode / Light mode toggle
* Add APIs to Favorites
* Pagination
* Loading indicator
* Responsive design for mobile, tablet, and desktop

---

## API Used

Public APIs Directory
https://api.publicapis.org/entries

This API provides a list of free public APIs including their name, description, category, authentication type, and HTTPS support.

## How to Run the Project

1. Clone the repository
2. Open the project folder
3. Open index.html in your browser
4. Make sure you have an internet connection for API requests

---

## Core Functionalities

* Fetch API data using JavaScript
* Render API data dynamically on the webpage
* Implement search using Array filter()
* Implement sorting using Array sort()
* Implement filtering using Array filter()
* Store favorites using Local Storage
* Responsive layout using CSS
